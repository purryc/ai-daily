# AI Daily

AI Daily is a bilingual GitHub Pages archive for daily AI soft/hardware product briefs.

The public site is plain static HTML, not a Slidev SPA. This avoids GitHub Pages subpath routing issues and keeps image-heavy evidence cards readable.

- Chinese homepage: `./index.html`
- English homepage: `./en/index.html`
- Daily Chinese issue: `./YYYY-MM-DD/zh/`
- Daily English issue: `./YYYY-MM-DD/en/`
- Daily source ledger: `./YYYY-MM-DD/sources.md`
- Issue data: `./data/issues.json`
- Renderer: `npm run build`
- Checks: `npm run check`
